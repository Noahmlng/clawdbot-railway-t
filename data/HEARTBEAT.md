# HEARTBEAT

Default behavior: reply `HEARTBEAT_OK`.

Only do more when the current task explicitly sets up a recurring check or there is a clearly actionable, current issue tied to work already in progress.

Rules:

- Stay quiet if nothing urgent or decision-worthy happened.
- Do not start broad self-healing, surprise work, or memory rewrites from a heartbeat alone.
- Keep any heartbeat notification concise and task-linked.
